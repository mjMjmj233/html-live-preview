# HTML 实时预览与组件定位工具

VS Code 插件，提供实时 HTML 预览，支持两种工作模式：
- **浏览模式**：正常浏览网页，完整交互体验
- **定位模式**：点击页面任意元素，自动跳转到对应源代码位置

## 功能特性

- **实时预览**：编辑 HTML 文件时自动更新预览
- **浏览模式**：正常浏览网页，链接、按钮、表单均可交互
- **定位模式**：点击元素自动在编辑器中高亮并跳转到源代码位置
- **源码映射**：精确建立 DOM 元素与 HTML 源码行号的对应关系
- **路径解析**：自动处理相对路径的 CSS、JS、图片等资源引用
- **进度显示**：顶部进度条实时展示大文件的处理进度

## 使用方法

### 打开预览
1. 在编辑器中打开 HTML 文件
2. 点击编辑器工具栏的预览图标，或
3. 执行命令：`HTML 预览：打开 HTML 实时预览`

### 切换模式
- 点击预览面板工具栏的**浏览模式 / 定位模式**按钮
- 或使用快捷键：`Ctrl+Shift+I` / `Cmd+Shift+I`
- 或执行命令：`HTML 预览：切换浏览/定位模式`

### 定位模式
1. 切换到**定位模式**
2. 鼠标悬停在元素上查看标签名和源码行号
3. 点击任意元素，自动在编辑器中高亮跳转到对应位置
4. 按 `ESC` 退出定位模式

### 浏览模式
- 正常浏览网页
- 所有交互自然透传

## 工作原理

插件解析 HTML 文件，为每个 DOM 元素注入唯一的 `data-comp-id` 标识，并建立元素与源代码行号的映射关系。在定位模式下点击元素时：

1. 捕获点击事件并识别目标元素
2. 读取元素的 `data-comp-id`
3. 查询映射表获取对应的源码行号范围
4. 打开编辑器并跳转到精确位置，带有高亮动画
